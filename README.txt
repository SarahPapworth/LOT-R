When accessed through RStudio, this code (app.R) can be used to run a ShinyApp where you can take the Revised Life Orientation Test and see the result.
To use, make sure you have installed the shiny package in RStudio.
You should then see a 'Run App' button when the code is open in RStudio.
Click this button and the app will open in a new window.

You can also use the app here:
https://sarahpapworth.shinyapps.io/LOTR/

This app was developed to accompany the paper:
Increased dispositional optimism in conservation professionals
which can be found here: https://link.springer.com/article/10.1007/s10531-018-1665-0
